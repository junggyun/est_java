## 변수

선언과 초기화
```java
public class Var {
    public static void main(String[] args) {
        int a; // 변수 선언
        a = 10; // 변수 초기화
    }
}
```
Long타입 초기화시 int의 범위를 넘어갈 경우 값 뒤에 L을 붙여야함
